# Kali Recovery Toolkit (KRT)

> Repair • Backup • Diagnose • Recover

## Overview
KRT is a professional open-source Bash toolkit focused on Linux recovery, UEFI, GRUB, EFI partitions, diagnostics and preventive maintenance.

## Philosophy
> Understand first.
>
> Backup second.
>
> Repair third.
