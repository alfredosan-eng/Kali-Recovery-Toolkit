# Recovery Module

## Purpose

The Recovery Module compares the current system against the latest snapshot
generated by the Backup Module.

Current implementation compares:

- Kernel
- Hostname
- Distribution
- Architecture

Future versions will compare:

- Memory
- Disk
- Installed Packages
- EFI
- GRUB
- Filesystems

The Recovery Module never modifies the system.

It only analyzes differences.
