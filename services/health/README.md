# Health Service

## Purpose

The Health Service evaluates the health status of system components.

It does not collect information.

It only evaluates collected values.

## Responsibilities

- Evaluate disk usage
- Evaluate memory
- Evaluate kernel
- Evaluate distribution
- Return standardized status values

## Status Levels

PASS

WARNING

ERROR

CRITICAL

## Used By

- Doctor Module
- Backup Module (future)
- Recovery Module (future)
